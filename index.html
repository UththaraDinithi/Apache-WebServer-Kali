My Personal Web Server Project

Exploring Linux, Apache, and Web Development Hands-On

Project Overview

This project demonstrates the setup of a fully functional Apache web server on Kali Linux, hosting a personal HTML website. It helped me understand Linux commands, server configuration, and web page deployment.

Skills Practiced

  • Linux system administration and terminal commands
  • Web server management with Apache
  • HTML page creation and content structuring
  • Problem-solving and troubleshooting

Challenges Overcome

Initially, I faced issues with Apache service configuration and file permissions. I researched and implemented solutions, which taught me server troubleshooting and Linux file management.

Key Learnings

This project gave me a real understanding of how web servers work, basic networking, and the deployment process. It also improved my confidence in Linux and web technologies.
